# zomato_data_analysis
Data Analysis and Visualization using Python

📌Project Overview
This project focuses on analyzing Zomato restaurant data using Python. The objective is to explore restaurant ratings, votes, and different restaurant categories to gain meaningful insights through Exploratory Data Analysis (EDA) and Data Visualization.
📊 Objectives

The analysis answers key business questions such as:

Which type of restaurant is most preferred by customers?
How many votes do different restaurant types receive?
What ratings are most common among restaurants?
What is the average spending for couples ordering online?
Which mode (online/offline) gets higher ratings?
Which restaurant type gets more offline orders?

🛠️ Tools & Libraries Used
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

🔄 Data Processing
Handled missing values
Converted rating column into numeric format
Cleaned and structured dataset for analysis

📈 Key Insights
Majority of customers prefer specific restaurant types (based on analysis)
Online and offline ordering patterns differ in spending and ratings
Certain restaurant categories perform better in offline orders
Ratings distribution shows customer satisfaction trends

⭐ Outcome

This project demonstrates basic data cleaning, visualization, and business insight generation using real-world restaurant data.
